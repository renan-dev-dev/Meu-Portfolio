Sobre mim

Domino as principais tecnologias do front-end: HTML, CSS, JavaScript, React e Tailwind CSS, aplicando conceitos de estruturação semântica, responsividade, além de animações e interações para criar experiências mais dinâmicas e atrativas.

Tenho foco em UI/UX, buscando sempre desenvolver interfaces intuitivas, acessíveis e agradáveis, que proporcionem uma experiência positiva ao usuário em cada interação.

Utilizo JavaScript e React para desenvolver funcionalidades como manipulação de elementos na página, integração com APIs, efeitos visuais e componentes interativos. Com o Tailwind CSS, consigo estilizar interfaces de forma ágil e consistente, garantindo um design moderno e responsivo.
